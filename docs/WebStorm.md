* 全局安装ts-node：
```
npm install -g ts-node
```
  * 虽然可以用 ts-node Hello_Word.ts 来运行 .ts 文件，但是要执行文件中的部分代码时还会报上面的错误。
  * 可以在项目本地安装 ts-node 来解决报错的问题：
```
  npm install ts-node --save-dev
```
